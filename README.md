BloggerRenamer
==============

A basic sample showing how to use Blogger API v3 with OAuth with C# .net 4.0

Prerequisites
============
1. Visual Studio 2012
2. .net 4.5 installed
3. NuGet installed (https://www.nuget.org/)
4. Windows 7 and/or above
5. Downloaded Google API .net Client (https://code.google.com/p/google-api-dotnet-client/source/checkout)
